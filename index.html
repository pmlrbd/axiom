Coherence ≠ Usefulness

When evaluating LLMs for logical reasoning, we typically check if models avoid contradictions. A model shouldn't simultaneously claim that a statement is both true AND false.

The Hidden Failure Mode: A model can achieve perfect coherence by refusing to commit to either answer. If it never says "yes" with high confidence, it can never contradict itself.

This is vacuous coherence—the model appears consistent because it never takes a stance, not because it reasons correctly.

Our Solution: Measure both coherence AND commitment. A useful reasoner must be coherent while making decisive predictions.
Key Insight

The Commitment Score

A simple metric that exposes vacuous coherence

The Mathematical Foundation

c(φ) = pyes(φ) + pyes(¬φ)

When commitment c < 1, violation vneg = 0 regardless of reasoning quality

● c > 1: Over-commitment

Model assigns >100% probability mass. Results in contradictions.

● c < 1: Under-commitment

Model hedges by assigning low probability to both options. Zero utility.

● c ≈ 1: Ideal

Model commits fully but correctly—high on right answer, low on wrong.

Methodology

The CAAC Framework

A simple three-step protocol for commitment-aware evaluation

1

Two-Query Elicitation

For each premise-conclusion pair, ask: Is φ entailed? Is ¬φ entailed?

p_yes(φ) = softmax(log P(YES|Q_φ))
p_yes(¬φ) = softmax(log P(YES|Q_¬φ))
2

Compute Metrics

Calculate commitment and violation scores.

c(φ) = p_yes(φ) + p_yes(¬φ)
v_neg(φ) = max(0, c(φ) - 1)
3

3-Way Decision

Map to True/False/Uncertain predictions.

if p(φ) ≥ τ: TRUE
if p(¬φ) ≥ τ: FALSE
else: UNCERTAIN
